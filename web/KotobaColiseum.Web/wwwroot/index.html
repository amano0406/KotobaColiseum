使い方

  • キー未設定でも画面は開きます。
  • dynamic は敵キャラ・導入・画像プロンプトをその場生成します。
  • fixed は既存の固定敵で安定起動するモードです。
  • 保存後に「疎通確認」で OpenAI API と接続テストできます。
  • 画像生成が失敗しても、ゲーム本体はプレースホルダーで続行します。
