使い方

  • 設定はモーダルから開けます。ゲーム画面はスクロールしません。
  • dynamic は敵キャラ、前後ストーリー、画像プロンプトをその場生成します。
  • fixed は固定敵で安定起動するモードです。
  • 保存後に「疎通確認」で OpenAI API と接続テストできます。
  • 画像生成が失敗しても、ゲーム本体はプレースホルダーで続行します。
